# OWA Calendar Notifier

Tampermonkey userscript для браузерных уведомлений о предстоящих встречах в Outlook Web App (Exchange 2019 on-premises).

## Зачем

OWA не умеет отправлять push-уведомления о встречах. Этот скрипт работает в фоне пока открыта вкладка OWA и напоминает о встречах через браузерные уведомления.

## Возможности

- Polling календаря каждые 30 секунд через OWA service.svc (same-origin, без отдельной авторизации)
- Уведомление за 15 минут до встречи и за 5 секунд до начала
- Звуковой сигнал при каждом уведомлении
- Клик по уведомлению открывает ссылку на встречу (Teams, Zoom, Google Meet, KTalk)
- Уведомление о потере связи, если нет подключения более 5 минут
- Floating-виджет с информацией о ближайшей встрече, статусом и логом

## Установка

1. Установить [Tampermonkey](https://www.tampermonkey.net/) в браузер
2. Создать новый скрипт и вставить содержимое `owa-calendar-notifier.user.js`
3. Открыть OWA — скрипт запустится автоматически
4. Разрешить уведомления в браузере при первом запуске

## Настройки

В начале скрипта можно изменить:

```js
const POLL_INTERVAL = 30_000;   // интервал проверки (мс)
const NOTIFY_BEFORE_MIN = 15;   // за сколько минут уведомлять
const CALENDAR_WINDOW_MIN = 60; // окно просмотра календаря (мин)
```

## Совместимость

- Exchange 2019 on-premises с OWA
- Chrome, Edge, Firefox с установленным Tampermonkey
- `@match https://*/owa/*` — работает на любом OWA-сервере
